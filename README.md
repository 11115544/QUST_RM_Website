# 青岛科技大学 YKTKEMIAO 战队网站

基于 Python Flask 的**战队展示与内容管理**网站，面向 RoboMaster 机甲大师赛事，用于战队简介、荣誉展示、战队风采、新人招募与赞助合作，支持 Markdown 笔记、图片与视频展示、在线编辑与上传。

**板块结构：**

- **战队简介**：发展历程（队史与建队）、组织架构
- **荣誉展示**：赛事荣誉（按赛季）
- **战队风采**：媒体素材（照片与视频、核心队员介绍）
- **新人招募**：招募说明与报名方式
- **赞助合作**：赞助权益与联系方式

## 如何运行（输入网址即可打开）

### 方式一：双击启动（推荐）

1. **首次使用**需先安装依赖（在项目文件夹里打开终端执行一次即可）：
   ```bash
   pip install -r requirements.txt
   ```
2. 之后每次要进网站时，**双击运行 `启动网站.bat`** 即可。
3. 约 1～2 秒后浏览器会自动打开 **http://127.0.0.1:5001**；若未自动打开，可手动在浏览器输入该地址。
4. 关闭弹出的黑色命令行窗口，或在该窗口按 `Ctrl+C`，即可停止网站。

### 方式二：在终端里运行

```bash
cd "C:\Users\ASUS\Desktop\team_wibsite\QUST_RM_Website"
pip install -r requirements.txt   # 仅首次需要

```

### 在浏览器中打开

- **本机访问**：**http://127.0.0.1:5001**（用 `启动网站.bat` 时通常会自动打开）
- **同一局域网内其他设备**：先查本机 IP（如 `192.168.1.100`），在浏览器输入 **http://本机IP:5001**

## 网站结构

- **汇总**：首页，进入各板块（战队简介、荣誉、风采、招募、赞助）。
- **各板块**：页内按**分类**列出**子页**，点击子页进入该页内容。
- **子页**：显示该页的 Markdown 内容；点击「编辑」可在线修改、**粘贴图片**、**上传视频/文件**，上传后自动插入到内容中。
- **管理**：增删改板块、分类、子页；子页正文仍在各子页的「编辑」中修改。

## 如何维护与扩展

### 编辑内容与上传

- **在线编辑**：进入任意板块 → 点击某**子页** → 点击「编辑」。在编辑页可写 Markdown，**粘贴图片**（Ctrl+V）或点击「图片 / 视频 / 文件」上传，上传后会自动插入到光标处（图片为 `![](...)`，视频为 `<video>`，其他为链接）。
- **内容存放**：按 **板块 id / 分类 id / 子页 id** 存为 `content/板块/分类/子页.md`，例如 `content/about/history/team.md`。
- **上传文件存放**：上传的图片、视频、PDF 等保存在 `static/uploads/板块/分类/子页/` 下，按板块与分类区分。

### 新增板块或子页

- 在 **「管理」页** 中可添加板块、分类、子页；或直接编辑 **`data/sections.json`** 仿照现有项添加 `id`、`name`、`brief`、`icon` 及 `categories` 列表。
- 内容文件路径：`content/板块id/分类id/子页id.md`，首次在编辑页保存时自动创建。

## 目录结构概览

```
QUST_RM_Website/
├── app.py              # 主程序，运行它即可启动网站
├── requirements.txt    # 依赖
├── 启动网站.bat        # 双击启动（推荐）
├── README.md           # 本说明
├── data/
│   └── sections.json   # 板块/分类/子页配置（管理页可改）
├── content/            # 按 板块/分类 存放 .md
│   └── 板块id/分类id/子页id.md
├── static/
│   ├── css/
│   └── uploads/        # 在线编辑时上传的图片/视频/文件（按板块/分类/子页分子目录）
└── templates/
    ├── base.html, index.html, section.html, subject.html, edit_subject.html, manage.html
```

按上述步骤运行后，在浏览器输入 **http://127.0.0.1:5001** 或 **http://本机IP:5001** 即可打开战队网站。
