# 新人招募

![迎新海报](/static/images/recruit-poster.png)

招募周期、各组别（机械、电控、算法、运营、宣传）招募要求。

- **机械组**：对机械设计、加工与装配感兴趣
- **电控组**：电路、嵌入式、C/C++ 或 Python
- **算法组**：视觉、决策、仿真或机器学习
- **运营组**：物资、经费、文档与协调
- **宣传组**：招新宣传、摄影与新媒体

**报名方式**：请通过战队官方渠道（QQ 群、公众号等）获取报名表或联系负责人。报名人员可后续登录查看审核进度。

当前招募阶段：由管理员在本文中更新（未开始 / 招募中 / 已结束）。
